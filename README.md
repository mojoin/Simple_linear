# 软件功能简介

本软件基于Qt框架实现了一个交互式数据可视化视图组件，专用于展示散点图、绘制回归直线并显示相关统计参数。核心功能围绕`GraphicsView`类构建，适用于科学计算、数据分析等场景。

## 主要功能

### 1. **坐标系绘制**
   - 自动绘制带网格线的笛卡尔坐标系，包括X轴和Y轴。
   - 支持刻度标记和原点标识，网格线以浅灰色虚线显示。
   - 坐标刻度根据视图缩放动态调整，确保可读性。

### 2. **数据点可视化**
   - 支持导入数据点集（`QVector<QPointF>`），并在坐标系中以红色填充的蓝色椭圆标记。
   - 每个数据点旁显示其坐标标签（格式为`(x, y)`），数据点坐标在内部缩放100倍以适应视图。
   - 数据点集可通过`setDataPoints`方法动态更新。

### 3. **回归分析**
   - 根据线性回归参数（斜率`a`和截距`b`）绘制红色回归直线。
   - 自动计算并显示回归方程（格式：`y = ax + b`）。
   - 在数据点中心附近展示统计参数：
     - 数据点数（`n`）
     - 相关系数（`r`）
     - 残差平方和（`sse`）
   - 通过`setRegressionLine`方法设置回归参数，支持动态更新。

### 4. **交互式视图**
   - 支持鼠标拖动平移视图（ScrollHandDrag模式）。
   - 滚轮缩放功能：向上滚动放大（缩放因子1.1），向下滚动缩小（缩放因子0.9）。
   - 视图中心可重置到原点（`resetView`方法）。
   - 优化性能：启用背景缓存和最小视口更新。

### 5. **清理与重置**
   - 一键清除所有数据点和回归线（`clearGraphics`方法）。
   - 视图缩放状态可重置（`resetView`方法）。

## 代码关键特性
- **坐标系范围**：场景默认设置为大范围（-10000到10000），适应不同数据尺度。
- **坐标缩放处理**：输入数据点自动缩放100倍，显示时还原真实值。
- **抗锯齿渲染**：确保图形平滑显示。
- **事件处理**：重写`wheelEvent`实现缩放，`drawBackground`和`drawForeground`分离绘制逻辑。
