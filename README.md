# Simple Linear Regression Application

这是一个基于Qt框架开发的一元线性回归分析应用程序。

## 功能特点

- 数据可视化展示
- 线性回归计算
- 图形化结果展示

## 技术栈

- Qt框架
- C++编程语言
- Eigen数学库(版本5.0.0)

## 使用说明

1. 编译项目:
   ```bash
   qmake Simple_linear.pro
   make
   ```
   
2. 运行程序

## 项目结构

- `mainwindow.*`: 主窗口实现
- `graphicsview.*`: 图形视图实现
- `eigen-5.0.0/`: Eigen数学库源码
